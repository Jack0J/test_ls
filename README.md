# Система скидок

Профессиональное представление системы скидок с четкой структурой и акцентом на данные.

## Особенности
- Четкое разделение на розничных и оптовых покупателей
- Табличное представление данных для быстрого восприятия
- Минималистичный, но структурированный дизайн
- Адаптивная верстка
- Без излишних визуальных эффектов

## Файлы
- `index.html` — основная страница
- `style.css` — стили

## Использование
Откройте `index.html` в браузере или разместите на GitHub Pages.
