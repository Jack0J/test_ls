# Система скидок

Минималистичное представление системы скидок для B2B и розничных покупателей.

## Файлы
- `index.html` — основная страница
- `style.css` — базовые стили

## Как посмотреть
1. Откройте `index.html` в браузере
2. Или загрузите на GitHub Pages

## Особенности
- Чёрно-белая цветовая схема
- Минималистичный дизайн
- Адаптивная вёрстка
- Без зависимостей
