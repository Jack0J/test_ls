# Система скидок

Система скидок для оптовых и розничных покупателей с полным текстом условий, контактами и формой обратной связи.

## Особенности
- Полный оригинальный текст условий
- Выделенный блок для спортивных школ
- Контакты для связи
- Форма обратной связи
- Компактный формат
- Ограниченная цветовая палитра
- Адаптивный дизайн

## Файлы
- `index.html` — основная страница
- `style.css` — стили

## Использование
Откройте `index.html` в браузере или разместите на GitHub Pages.
